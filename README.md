# BetterActionBar

Fabric client mod that adds newline support for actionbar messages containing `\n`.

为 actionbar 消息内容添加换行支持的Fabric客户端模组。

## Usage / 用法
Install for your game version, it just works.

根据你的游戏版本安装即可。
