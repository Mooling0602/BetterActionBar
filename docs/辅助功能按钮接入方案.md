# 实现方案
这是在“辅助功能”设置页追加 BetterActionBar 按钮的完整实现方案。

内容整理自外部聊天记录，已经过可行性验证，现按原样记录于此。部分示例代码为其他项目的代码，使用时需根据实际情况进行修改。

> 仅适用于26.2-snapshot-*版本。

## 按钮
```java
private void addOptionsWithEasySort(CallbackInfo ci) {
    ci.cancel(); // 取消原方法执行

    AccessibilityOptionsScreen screen = (AccessibilityOptionsScreen) (Object) this;
    OptionsList list = ((OptionsSubScreenAccessor) screen).getList();

    // 调用原类的 options 方法获取所有选项
    OptionInstance<?>[] activeOptions = getOptions(screen);

    // 创建 EasySort 按钮
    Button easySortButton = Button.builder(
    Component.translatable("easySort.options.button"),
    (btn) -> {
    EasySortClient.LOG.info(msg: "EasySort: Click EasySort Button Clicked!");
    }
    ).width(150).build();
}
```
先`ci.cancel();`取消掉原始函数的执行

然后获取原始函数的screen和optionlist

`OptionInstance<?>[] activeOptions = getOptions(screen);`，用它原来的选项实例获取这个screen里的所有Option

然后创建你自己的按钮，for遍历screen里所有option的那个数组，处理到最后一位停止，也就是n-1位

然后addSmall函数是添加一行两个按钮，左侧按钮必须提供，右侧按钮可以是null，所以我们不能直接用addSmall来创建现有左侧按钮的右侧按钮

把之前留下来的第n位options和你自己的按钮组合在一起，传给addSmall，把第n位和你的按钮一起处理了
```java
AbstractWidget lastWidget = activeOptions[lastIndex].createButton(
    ((OptionsSubScreenAccessor) screen).getOptions()
);
```

第n位的option这样获取它原来的组件
```java
private OptionInstance<?>[] getOptions(AccessibilityOptionsScreen screen) {
    try {
        java.lang.reflect.Method method = AccessibilityOptionsScreen.class.getDeclaredMethod(
            "options", net.minecraft.client.Options.class
        );
        method.setAccessible(true);
        return (OptionInstance<?>[]) method.invoke(
            screen, ((OptionsSubScreenAccessor) screen).getOptions()
        );
    } catch (Exception e) {
        EasySortClient.LOG.error("Failed to get options", e);
        return new OptionInstance[0];
    }
}
```

这是获取optionlist和options的工具接口
```java
package neko.shulker.easysort.client.mixin;

import org.spongepowered.asm.mixin.Mixin;
import org.spongepowered.asm.mixin.gen.Accessor;

import net.minecraft.client.Options;
import net.minecraft.client.gui.components.OptionsList;
import net.minecraft.client.gui.screens.options.OptionsSubScreen;

@Mixin(OptionsSubScreen.class)
public interface OptionsSubScreenAccessor {

    @Accessor("list")
    OptionsList getList();

    @Accessor("options")
    Options getOptions();
}
```
